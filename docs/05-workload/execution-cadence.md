# Execution Cadence

## Daily loop
1. Resume with `/kickoff-v2`
2. Pick exactly one active workstream
3. Produce at least one new artifact, fit, trace, or gate update
4. Record evidence and session handoff

## Weekly checkpoint
- update gate status
- summarize what changed in evidence index
- check whether any second-order work started too early
- prune stale assumptions

## Anti-patterns
- writing broad summaries without new evidence
- mixing trace design and report prose in the same long session
- discussing prediction before trace + timing + simulator exist
